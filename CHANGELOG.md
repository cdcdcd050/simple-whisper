# Changelog

## [1.1.2] - 2026-03-24

### Changed
- **소리 알림 개선**: 팝업이 닫혀있을 때만 소리 재생, 열려있으면 소리 없음
  - 같은 상대의 연속 귓속말은 90초간 소리 무시
  - 전투 중 팝업이 안 열리는 설정이면 상대별 1회 소리 재생
  - 전투 종료 후 자동 열기 시에도 소리 재생 (90초 디바운스 적용)
  - 발신 시 소리 없음
- **레포 구조 변경**: CurseForge 호환을 위해 .toc 파일을 레포 루트로 이동

### Fixed
- BNet 귓속말 자동 열기 시 대화 선택이 안 되던 버그 수정
- 전투 종료 시 잘못된 변수 참조 수정

## [1.1.1] - 2026-03-24

### Added
- **읽음 구분선**: 창을 닫거나 대화를 전환한 후 새 메시지가 도착하면 `— 여기까지 읽음 —` 구분선을 표시하여 이전 대화와 새 메시지를 시각적으로 구분
  - 창 닫기, 대화 전환, 로그아웃 시 마지막 읽은 위치를 자동 저장
  - 새 메시지가 없으면 구분선 미표시
  - 대화 삭제 및 전체삭제 시 읽음 위치 정리
  - 복사 기능에도 읽음 구분선 반영
  - 다국어 지원 (영어: "Read up to here" / 한글: "여기까지 읽음")

### Changed
- **날짜 구분선에 시간 추가**: `— 2026-03-24 —` → `— 2026-03-24 19:30:00 —` 형태로 해당 시점의 시간까지 표시
- **구분선 중복 방지**: 읽음 구분선과 날짜 구분선이 동시에 발생하면 날짜+시간 구분선만 표시
