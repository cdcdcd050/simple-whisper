# Changelog

## [1.2.4] - 2026-03-24

### Added
- **소리 선택지 확장**: 5종으로 확대 — 귓속말/경매장/초대(SoundKit)/BNet(SoundKit)/커스텀
- **SoundKit ID 지원**: 파일 경로 외에 WoW SoundKit ID로도 소리 재생 가능

## [1.2.3] - 2026-03-24

### Changed
- **알림 소리 채널 변경**: Master → SFX (효과음) 채널로 변경, WoW 효과음 볼륨으로 조절 가능

## [1.2.2] - 2026-03-24

### Changed
- **알림 소리 디바운스 단축**: 같은 상대 중복 소리 방지 간격 90초 → 30초
- **발신 시 알림 소리 추가**: 귓속말 발신(일반/BNet) 시 최초 1회 소리 재생
- **수신/발신 디바운스 분리**: 수신·발신 각각 독립적으로 30초 디바운스 적용

## [1.2.1] - 2026-03-24

### Added
- **CurseForge 자동 패키징**: `.pkgmeta` 추가, GitHub 태그 push 시 CurseForge 자동 배포

## [1.1.5] - 2026-03-24

### Added
- **입력창 슬래시 명령어 지원**: SimpleWhisper 입력창에서 `/`로 시작하는 WoW 명령어 실행 가능 (`/invite`, `/who`, `/swsw` 등)

## [1.1.4] - 2026-03-24

### Changed
- **이름 목록 순서 개선**: 발신/답신/이름 클릭 시 목록 순서 변경 안 됨, 새 귓 수신(다른 상대)만 맨 위로 이동
- **안 읽음 중복 BumpName 방지**: 이미 안 읽음이 있는 상대는 추가 귓이 와도 목록 위치 유지
- **안 읽음 빨간 바 표시**: 안 읽은 대화 좌측에 빨간 바 표시 (선택 시 금색 바 우선)
- **테스트 명령어 추가**: `/swsw t2`, `/swsw t3` (TestPlayer2, TestPlayer3 시뮬레이션)

### Fixed
- 외부 이름 클릭 시 nameList에 추가되지 않던 버그 수정

## [1.1.3] - 2026-03-24

### Changed
- **읽음 구분선 개선**: 3줄 → 1줄로 간결하게 변경, 색상 회색 통일
- **발신 시 구분선 표시**: 발신으로 팝업 열 때도 이전 대화 끝에 구분선 미리 표시

## [1.1.2] - 2026-03-24

### Changed
- **소리 알림 개선**: 팝업이 닫혀있을 때만 소리 재생, 열려있으면 소리 없음
  - 같은 상대의 연속 귓속말은 90초간 소리 무시
  - 전투 중 팝업이 안 열리는 설정이면 상대별 1회 소리 재생
  - 전투 종료 후 자동 열기 시에도 소리 재생 (90초 디바운스 적용)
  - 발신 시 소리 없음
- **레포 구조 변경**: CurseForge 호환을 위해 .toc 파일을 레포 루트로 이동

### Fixed
- BNet 귓속말 자동 열기 시 대화 선택이 안 되던 버그 수정
- 전투 종료 시 잘못된 변수 참조 수정

## [1.1.1] - 2026-03-24

### Added
- **읽음 구분선**: 창을 닫거나 대화를 전환한 후 새 메시지가 도착하면 `— 여기까지 읽음 —` 구분선을 표시하여 이전 대화와 새 메시지를 시각적으로 구분
  - 창 닫기, 대화 전환, 로그아웃 시 마지막 읽은 위치를 자동 저장
  - 새 메시지가 없으면 구분선 미표시
  - 대화 삭제 및 전체삭제 시 읽음 위치 정리
  - 복사 기능에도 읽음 구분선 반영
  - 다국어 지원 (영어: "Read up to here" / 한글: "여기까지 읽음")

### Changed
- **날짜 구분선에 시간 추가**: `— 2026-03-24 —` → `— 2026-03-24 19:30:00 —` 형태로 해당 시점의 시간까지 표시
- **구분선 중복 방지**: 읽음 구분선과 날짜 구분선이 동시에 발생하면 날짜+시간 구분선만 표시
