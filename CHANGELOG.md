# Changelog

## [1.1.1] - 2026-03-24

### Added
- **읽음 구분선**: 창을 닫거나 대화를 전환한 후 새 메시지가 도착하면 `— 여기까지 읽음 —` 구분선을 표시하여 이전 대화와 새 메시지를 시각적으로 구분
  - 창 닫기, 대화 전환, 로그아웃 시 마지막 읽은 위치를 자동 저장
  - 새 메시지가 없으면 구분선 미표시
  - 대화 삭제 및 전체삭제 시 읽음 위치 정리
  - 복사 기능에도 읽음 구분선 반영
  - 다국어 지원 (영어: "Read up to here" / 한글: "여기까지 읽음")

### Changed
- **날짜 구분선에 시간 추가**: `— 2026-03-24 —` → `— 2026-03-24 19:30:00 —` 형태로 해당 시점의 시간까지 표시
- **구분선 중복 방지**: 읽음 구분선과 날짜 구분선이 동시에 발생하면 날짜+시간 구분선만 표시
