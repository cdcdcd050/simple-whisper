# SimpleWhisper (귓속말)

간단한 WoW 귓속말(Whisper) 메신저 애드온

기존 귓속말 애드온들이 과도하게 복잡하여, 핵심 기능만 깔끔하게 구현한 애드온입니다.

![Interface](https://img.shields.io/badge/Interface-Classic~Retail-blue)
![Version](https://img.shields.io/badge/Version-1.1.0-green)

## 주요 기능

- **귓속말 전용 메신저 UI** — 대화 상대 목록 + 채팅 창 분리 레이아웃
- **BNet(배틀넷) 귓속말 지원** — 일반 귓속말과 BNet 귓속말 통합 관리
- **수신 알림** — 소리 알림 (3종 선택), 자동 창 열기
- **전투 모드** — 전투 중 귓속말 처리 방식 선택 (즉시 / 종료후 / 안열기)
- **채팅창 연동** — 채팅창의 이름 클릭 시 SimpleWhisper로 대화 열기
- **기본 채팅창 숨기기** — 귓속말을 기본 채팅창에서 숨기고 SimpleWhisper에만 표시
- **/who 자동 조회** — 대화 상대 선택 시 레벨, 직업, 길드 정보 표시
- **직업 색상 표시** — 이름 목록에 WoW 클래스 색상 적용
- **메모 기능** — 대화 상대별 메모 저장
- **미니맵 버튼** — 안 읽은 메시지 수 배지 표시
- **LDB 지원** — 데이터 브로커 호환 (Arcana 등)

## 설치

1. [Releases](https://github.com/cdcdcd050/simplewhisper/releases)에서 최신 `SimpleWhisper.zip` 다운로드
2. 압축 해제하여 `SimpleWhisper` 폴더를 WoW 애드온 디렉토리에 복사
   - `World of Warcraft/_버전_/Interface/AddOns/SimpleWhisper/`
3. 게임 재시작 또는 `/reload`

## 사용법
